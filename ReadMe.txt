This is a simplified template of GPU-DAEMON to guide users
about the flow of GPU-DAEMON. Aim of this template-code is 
to provide users with a helpful starting point. Different 
functions in the template can then be replaced by user defined
data specific functions.

this template contains a very simple example of processing array 
based data. For more complex and finished examples of this template
please refer to the GPU-ArraySort and G-MSR codes.
